Documentação da API

Rota: Listagem de Proposições

GET /listar

Descrição

Retorna uma lista de proposições com base nos parâmetros fornecidos.

Parâmetros

Parâmetro Descrição Padrão
autor ID do autor ID do autor padrão
ano Ano da proposição Ano atual
tipo Tipo da proposição (ex: PL) PL
arquivada Filtra proposições arquivadas (0 ou 1) 0
itens Quantidade de itens por página 10
pagina Número da página 1
ordenarPor Campo para ordenar (ex: proposicao_id) proposicao_id
ordem Ordenação (ASC ou DESC) DESC
autoria_unica Filtra proposições com autoria única (1 ou 0) 0
busca Termo para buscar na ementa Não há busca aplicada

Exemplo de Requisição

GET /listar?autor=123&ano=2023&tipo=PL&arquivada=0&itens=10&pagina=1&ordenarPor=proposicao_id&ordem=DESC

Rota: Medidas Provisórias

GET /medidas-provisorias

Descrição

Retorna uma lista de medidas provisórias publicadas em um determinado ano, com detalhes sobre as emendas e autores, incluindo as de autoria específica.

Parâmetros

Parâmetro Descrição Padrão
ano Ano das medidas provisórias 2024
itens Quantidade de itens por página 10
pagina Número da página 1

Exemplo de Requisição

GET /medidas-provisorias?ano=2024&itens=10&pagina=1

Rota: Buscar Proposição Principal

GET /buscar-principal

Descrição

Retorna os dados principais de uma proposição a partir de seu ID, incluindo informações sobre emendas e autores.

Parâmetros

Parâmetro Descrição Padrão
id ID da proposição Não há valor padrão

Exemplo de Requisição

GET /buscar-principal?id=12345

Rota: Buscar Apensados do Gabinete

GET /buscar-apensados-gabinete

Descrição

Retorna as proposições apensadas ao gabinete, ou seja, aquelas que estão relacionadas com a proposição principal, com base no ID da proposição.

Parâmetros

Parâmetro Descrição Padrão
id ID da proposição principal Não há valor padrão

Exemplo de Requisição

GET /buscar-apensados-gabinete?id=12345
